# SpringBoot 2.x笔记

## 概述

### 环境要求

- Jdk1.8+ 
- Maven 3.3+

- springboot 2.x ，大部分案例采用 2.4.3版本
- IDEA2020
- redis
- mysql8.x（兼容5.7版本）

### 项目说明
敏感数据建议保存在 application-mysecret.yml  application-mysecret.yaml  application-mysecret.properties

github 上传时会忽略所有项目中的该文件
